  • 팝업 ID 기준으로 호출하기때문에 팝업끼리 ID가 중복되면 안됩니다. 중복일 경우 변경해주시면 됩니다.
  • 팝업의 ID는 임시ID 입니다. 편한대로 변경해도 됩니다.
  • 팝업의 형태가 다르다 하여도 클래스로 구분되기때문에 모두 아래와 같은 방법으로 사용하시면됩니다.

오픈 방법

함수로 오픈

						Dialog.open('#fullDialog');
Dialog.open('해당팝업ID');
  • 띄울려는 팝업의 ID를 넣어주면 됩니다.

클래스로 오픈

						모달팝업
<a href="해당팝업ID" class="btn bs1 bc1 ui_dialog_open">모달팝업</a>
  • 클래스로 ui_dialog_open를 넣어주시고 href속성에 팝업의 ID를 넣어주면 됩니다.
  • ui_dialog_open클래스는 해당 용도가 아니면 에러를 발생할수 있으니 안쓸때는 삭제해주세요.

닫기 방법

함수로 닫기

						Dialog.close('#fullDialog');
Dialog.close('해당팝업ID');
  • 숨기려는 팝업의 ID를 넣어주면 됩니다.

클래스로 닫기

						팝업닫기
<a href="해당팝업ID" class="btn bs1 bc1 ui_dialog_close">팝업닫기</a>
또는 (닫으려는 팝업안에 있는 버튼이라면 클래스만 붙여도 작동)
<a href="#" class="btn bs1 bc1 ui_dialog_close">팝업닫기</a>
  • 클래스로 ui_dialog_close를 넣어주시고 href속성에 팝업의 ID를 넣어주면 됩니다.
  • 만약 닫으려는 팝업 안에 있는 버튼이라면 href속성에 팝업의 ID를 넣지 않아도 됩니다.
  • ui_dialog_close클래스는 해당 용도가 아니면 에러를 발생할수 있으니 안쓸때는 삭제해주세요.

  • 퍼블팀에서 작업한 전체선택 기능은 '전체선택'의 .all_chk 클래스 와 '리스트'의 .item_chk 클래스 기준으로 작동합니다.
  • 따로 조건문등을 사용하여 전체선택 기능을 별도로 구현하실 경우 해당 클래스들(.all_chk,.item_chk)을 삭제하시면 됩니다.

퍼블팀 '전체선택'기능 제거: .all_chk와 .item_chk 클래스 삭제 (별도 스크립트)

계약 목록

별도 구현시 샘플 스크립트

						//전체선택 checkbox
						$('[name=allChk2]').click(function(){
							if($(this).prop('checked')){
								//전체선택 시킬때
								$(this).siblings('label').changeTxt('선택','해제');
								$('[name=product]').prop('checked',true).change();
							}else{
								//전체해제 시킬때
								$(this).siblings('label').changeTxt('해제','선택');
								$('[name=product]').prop('checked',false).change();
							}
						});

						//리스트 선택 checkbox
						$('[name=product]').click(function(){
							var $chkLength = $('[name=product]').length,
								$chkedLength = $('[name=product]:checked').length;
							if($(this).prop('checked')){
								if($chkLength == $chkedLength){
									$('[name=allChk2]').prop('checked',true).siblings('label').changeTxt('선택','해제');
								}
							}else{
								$('[name=allChk2]').prop('checked',false).siblings('label').changeTxt('해제','선택');
							}
						});
  • 리스트안에 체크박스 선택 시 모두 체크되었는지 확인해서 '전체선택'을 활성화 시켜야합니다.
  • '전체선택' 활성화될때 텍스트는 '전체해제'로 변경되어야합니다. 비활성화일때는 다시 '전체선택'으로~
  • '전체선택' 클릭 시 $('[name=product]').prop('checked',true).change(); 이렇게 뒤에 change()를 붙여서 리스트안에 글자가 블루컬러로 바뀌는지 확인해야합니다.
